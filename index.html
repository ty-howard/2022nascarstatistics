CSCE 242: Web Applications

Ty Howard

This class teaches the beginnings of web development.

Assignments

Assignment 1 - Basic HTML

This assignment is an introduction to HTML and features a comparison of the A330-900neo and Boeing 787-9 airplanes.

Assignment 2 - Basic CSS

This assignment is an introduction to CSS and featues a table with my favorite airplanes, details about them, and images.

Assignment 3 - Page Layout

This assignment is the fundamentals of webpage layout using flex, and features basic placeholders to practice implementing HTML elements with CSS.
